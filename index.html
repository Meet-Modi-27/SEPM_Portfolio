Shared Inventory

A Smart Way to Manage and Share Group Items

📌 Problem Statement

Managing shared items in group environments like hostels, student apartments, or clubs often leads to confusion. People lose track of what is available, what’s missing, and who owns what. This results in miscommunication, duplication, or under-utilization of shared resources.

🎯 Objective

To design and develop an Android app that simplifies personal and shared inventory management. It allows users to maintain their own list of items while providing an easy way to share and view others' inventories using a secure code.

💡 Proposed Solution

The app enables users to add, update, and delete items in their inventory, share it securely via a generated code, and view shared inventories. With Firebase integration and QR scanning, the app becomes a one-stop solution for collaborative inventory tracking.

🚀 Features

  • 📋 Add and edit items with name, quantity, and location
  • ❌ Delete items when no longer needed
  • 🔗 Share inventory using a unique access code
  • 👀 View other users’ shared inventories
  • 📱 Clean, simple UI with RecyclerView
  • 📷 QR Code scanning for quick access

🛠️ Technologies Used

  • Java (Android Studio)
  • Firebase Realtime Database
  • Firebase Authentication
  • XML (UI Layouts)
  • ZXing Library for QR Integration
