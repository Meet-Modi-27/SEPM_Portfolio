Shared Inventory

A Smart Way to Manage and Share Items

Problem Statement

In many group scenarios such as hostels, clubs, or shared apartments, people often struggle to manage shared items. Keeping track of what items are available, where they are, and who owns them becomes confusing and unorganized.

Objective

To create an Android application that allows users to manage their own inventory and also view inventories shared by others. The app helps reduce confusion, saves time, and ensures better coordination between users.

Proposed Solution

The Shared Inventory app provides a simple way for users to add, update, and delete items from their personal inventory. They can also generate a code and share it with others, allowing them to view the shared items. This builds a centralized and transparent system for tracking shared resources.

Features

  • Add items with name, quantity, and location
  • Edit or delete existing items
  • Generate a unique code to share your inventory
  • View other users' shared items by entering their code
  • Simple and user-friendly interface
  • QR code integration for fast item identification

Technologies Used

  • Android Studio (Java)
  • Firebase Realtime Database
  • Firebase Authentication
  • XML (for UI design)
